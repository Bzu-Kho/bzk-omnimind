# рЏњрЏЅрџ▓ рЏЪрЏЌрџЙрЏЂрЏЌрЏЂрџЙрЏъ рџ▓рЏЪрџ▒рЏѕрЏЪрџ▒рџерЏЈрЏЂрЏЪрџЙРёб - рЏірЏЂрЏЈрЏЂрЏЪ рџ╣рЏќрЏњ рџ▓рЏЪрџ▒рЏѕрЏЪрџ▒рџерЏЈрЏЂрџбрЏЪ

рџбрџЙ рЏірЏЂрЏЈрЏЂрЏЪ рџ╣рЏќрЏњ рЏЌрЏЪрЏърЏќрџ▒рџЙрЏЪ рџц рџ▒рЏќрЏірЏѕрЏЪрџЙрЏірЏЂрџбрЏќ рЏѕрџерџ▒рџе рЏџрџе рЏќрЏЌрЏѕрџ▒рЏќрЏірџе рЏњрЏЅрџ▓ рЏЪрЏЌрџЙрЏЂрЏЌрЏЂрџЙрЏъ рџ▓рЏЪрџ▒рЏѕрЏЪрџ▒рџерЏЈрЏЂрЏЪрџЙРёб, рЏќрЏірЏѕрЏќрџ▓рЏЂрџерЏџрЏЂрЏЅрџерЏърџе рЏќрџЙ рЏірЏЪрЏџрџбрџ▓рЏЂрЏЪрџЙрЏќрЏі рЏЈрЏќрџ▓рџЙрЏЪрЏџрЏЪрџирЏЂрџ▓рџерЏі рЏЂрџЙрЏЈрЏќрЏџрЏЂрџирЏќрџЙрЏЈрЏќрЏі рџц рЏѕрџ▒рЏЪрЏърџбрџ▓рЏЈрЏЪрЏі рЏърЏќ рЏЂрџЙрЏЈрЏќрЏџрЏЂрџирЏќрџЙрџ▓рЏЂрџе рџерџ▒рЏЈрЏЂрџарЏЂрџ▓рЏЂрџерЏџ рџ▓рЏЪрџЙ рЏѕрџ▒рЏЪрЏЈрЏќрџ▓рџ▓рЏЂрЏЪрџЙ рЏќрЏЈрЏЂрџ▓рџе.

## ­Ъџђ рџ▓рџерџ▒рџерџ▓рЏЈрЏќрџ▒рЏЂрЏірЏЈрЏЂрџ▓рџерЏі

- **рЏърЏЂрЏірЏќрЏюрЏЪ рџ▒рЏќрЏірЏѕрЏЪрџЙрЏірЏЂрџбрЏќ**: рЏЪрЏѕрЏЈрЏЂрЏЌрЏЂрЏЅрџерЏърЏЪ рЏѕрџерџ▒рџе рЏърЏЂрЏірЏѕрЏЪрЏірЏЂрЏЈрЏЂрџбрЏЪрЏі рЏЌрЏЪрџбрЏЂрЏџрЏќрЏі, рЏЈрџерЏњрЏџрЏќрЏЈрЏі рџц рЏърЏќрЏірџ▓рЏЈрЏЪрЏѕ
- **рџЙрџерџбрЏќрџирџерџ▓рЏЂрЏЪрџЙ рЏірџбрџерџбрЏќ**: рЏірЏЌрЏЪрЏЪрЏЈрџ║ рЏірџ▓рџ▒рЏЪрЏџрЏџ рџц рџЙрџерџбрЏќрџирџерџ▓рЏЂрЏЪрџЙ рЏЂрџЙрЏЈрџбрЏЂрЏЈрЏЂрџбрџе
- **рџерџЙрЏЂрЏЌрџерџ▓рЏЂрЏЪрџЙрЏќрЏі рЏЌрЏЪрЏърЏќрџ▒рџЙрџерЏі**: рЏќрџарЏќрџ▓рЏЈрЏЪрЏі рџбрЏЂрЏірџбрџерЏџрЏќрЏі рџерЏЈрџ▒рџерџ▓рЏЈрЏЂрџбрЏЪрЏі рџц рЏѕрџ▒рЏЪрџарЏќрЏірЏЂрЏЪрџЙрџерЏџрЏќрЏі
- **рџарЏЪрџ▒рЏЌрџбрЏџрџерџ▒рЏЂрЏЪ рЏърЏќ рџ▓рЏЪрџЙрЏЈрџерџ▓рЏЈрЏЪ**: рЏірЏЂрЏірЏЈрЏќрЏЌрџе рЏърЏќ рџ▓рЏЪрџЙрЏЈрџерџ▓рЏЈрЏЪ рЏЂрџЙрЏЈрЏќрџ▒рџерџ▓рЏЈрЏЂрџбрЏЪ рџ▓рЏЪрџЙ рџ▓рџерЏЈрЏќрџирЏЪрџ▒рЏЂрџерЏі рЏќрЏірЏѕрЏќрџ▓рЏЂрџарЏЂрџ▓рџерЏі
- **рЏірЏќрЏЪ рЏЪрЏѕрЏЈрЏЂрЏЌрЏЂрЏЅрџерЏърЏЪ**: рЏќрЏірЏЈрџ▒рџбрџ▓рЏЈрџбрџ▒рџе рЏірЏќрЏЌрџерџЙрЏЈрЏЂрџ▓рџе рџц рЏЌрЏќрЏЈрџерЏърџерЏЈрЏЪрЏі рџерЏѕрџ▒рЏЪрЏѕрЏЂрџерЏърЏЪрЏі
- **рЏѕрџ▒рЏЪрЏърџбрџ▓рЏЈрЏЪрЏі рЏърЏќрЏірЏЈрџерџ▓рџерЏърЏЪрЏі**: рЏірЏќрџ▓рџ▓рЏЂрЏЪрџЙ рЏърЏќрЏърЏЂрџ▓рџерЏърџе рџе рЏірџбрЏѕрЏќрџ▒ рЏърЏЪрЏњ4 рџерЏЂРёб рџц рЏЌрџерЏЈрџ▒рЏЂрџ┤-рџерЏЂРёб

## ­ЪЊІ рЏірЏќрџ▓рџ▓рЏЂрЏЪрџЙрЏќрЏі рЏърЏќрЏџ рЏірЏЂрЏЈрЏЂрЏЪ

### ­ЪЈа рЏЂрџЙрЏЂрџ▓рЏЂрЏЪ (рџ║рЏќрџ▒рЏЪ)
- рЏѕрџ▒рЏќрЏірЏќрџЙрЏЈрџерџ▓рЏЂрЏЪрџЙ рЏЂрЏЌрЏѕрџерџ▓рЏЈрџерџЙрЏЈрЏќ рЏърЏќ рЏњрЏЅрџ▓ рЏЪрЏЌрџЙрЏЂрЏЌрЏЂрџЙрЏъ рџ▓рЏЪрџ▒рЏѕрЏЪрџ▒рџерЏЈрЏЂрЏЪрџЙРёб
- рЏџрЏџрџерЏЌрџерЏърџе рџе рЏџрџе рџерџ▓рџ▓рЏЂрЏЪрџЙ рЏѕрџ▒рЏЂрџЙрџ▓рЏЂрЏѕрџерЏџ
- рЏърЏЂрЏірЏќрЏюрЏЪ рџ▓рЏЪрџЙ рџирџ▒рџерЏърЏЂрЏќрџЙрЏЈрЏќ рЏЌрЏЪрЏърЏќрџ▒рџЙрЏЪ

### ­ЪЉЦ рџЙрЏЪрЏірЏЪрЏЈрџ▒рЏЪрЏі
- рџ║рЏЂрЏірЏЈрЏЪрџ▒рЏЂрџе рџц рџбрџерЏџрЏЪрџ▒рЏќрЏі рЏърЏќ рЏџрџе рЏќрЏЌрЏѕрџ▒рЏќрЏірџе
- рЏѕрџ▒рЏЪрЏърџбрџ▓рЏЈрЏЪрЏі рЏЂрџЙрЏірЏЂрџирџЙрЏЂрџе рЏърЏќрЏірЏЈрџерџ▓рџерЏърЏЪрЏі рџ▓рЏЪрџЙ рЏњрџерЏърџирЏќрЏі
- рЏќрЏірЏЈрџерЏърЏЂрЏірЏЈрЏЂрџ▓рџерЏі рЏърЏќрЏірЏЈрџерџ▓рџерЏърџерЏі (рЏѕрџ▒рЏЪрџцрЏќрџ▓рЏЈрЏЪрЏі, рџ▓рЏџрЏЂрЏќрџЙрЏЈрЏќрЏі, рЏќрџ┤рЏірЏѕрЏќрџ▒рЏЂрЏќрџЙрџ▓рЏЂрџе)
- рџ▓рЏЪрЏЌрЏѕрџ▒рЏЪрЏЌрЏЂрЏірЏЪ рЏќрЏЈрЏЂрџ▓рЏЪ рџц рЏѕрџ▒рЏЪрЏЈрЏќрџ▓рџ▓рЏЂрЏЪрџЙ рЏърЏќ рЏърЏќрџ▒рЏќрџ▓рџ║рЏЪрЏі рџ║рџбрЏЌрџерџЙрЏЪрЏі

### ­ЪЏа№ИЈ рЏѕрџ▒рЏЪрЏърџбрџ▓рЏЈрЏЪрЏі
- **рЏірџбрЏѕрЏќрџ▒ рЏърЏЪрЏњ4 рџерЏЂРёб**: рЏірЏЂрЏірЏЈрЏќрЏЌрџе рЏърЏќ рЏЂрџе рџ▓рЏЪрџЙ рЏѕрџ▒рЏЪрЏЈрЏќрџ▓рџ▓рЏЂрЏЪрџЙ рЏќрЏЈрЏЂрџ▓рџе рЏЂрџЙрЏЈрЏќрџирџ▒рџерЏърџе
- **рЏЌрџерЏЈрџ▒рЏЂрџ┤-рџерЏЂРёб**: рЏѕрЏџрџерЏЈрџерџарЏЪрџ▒рЏЌрџе рЏЌрџерЏЈрџ▒рЏЂрџ▓рЏЂрџерЏџ рЏърЏќ рЏЂрџе (рЏќрџЙ рЏърЏќрЏірџерџ▒рџ▒рЏЪрЏџрЏџрЏЪ)
- рЏЂрџЙрџарЏЪрџ▒рЏЌрџерџ▓рЏЂрЏЪрџЙ рЏірЏЪрЏњрџ▒рЏќ рЏџрЏЂрџ▓рЏќрџЙрџ▓рЏЂрџерЏі рЏЌрЏЂрЏЈ рџ▓рЏЪрџЙ рџ▓рЏџрџерџбрЏірџбрЏџрџерЏі рЏќрЏЈрЏЂрџ▓рџерЏі
- рџ▓рЏЪрЏЌрЏѕрџ▒рЏЪрЏЌрЏЂрЏірЏЪ рџ▓рЏЪрџЙ рЏќрЏџ рџбрЏірЏЪ рџ▒рЏќрЏірЏѕрЏЪрџЙрЏірџерЏњрЏџрЏќ рЏърЏќ рЏџрџе рЏЈрЏќрџ▓рџЙрЏЪрЏџрЏЪрџирЏЂрџе

- рџарЏЪрџ▒рЏЌрџбрЏџрџерџ▒рЏЂрЏЪ рЏърЏќ рџ▓рЏЪрџЙрЏЈрџерџ▓рЏЈрЏЪ рџ▓рЏЪрџЙ рџ▓рџерЏЈрЏќрџирЏЪрџ▒рЏЂрџерЏі рЏќрЏірЏѕрЏќрџ▓рЏЂрџарЏЂрџ▓рџерЏі
